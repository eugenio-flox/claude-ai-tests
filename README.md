# Claude AI Tests

Repositorio personal para documentar pruebas y casos de uso con Claude.

## Objetivo

El objetivo de este repositorio es recopilar ejemplos sencillos de uso de inteligencia artificial para mejorar tareas de productividad, organización, redacción y documentación.

## Casos de uso previstos

- Redacción y mejora de textos.
- Resumen de documentos.
- Organización de información.
- Creación de listas de tareas.
- Apoyo al estudio.
- Automatización básica de tareas administrativas.
- Generación de ideas y plantillas.

## Ejemplos de pruebas

### 1. Resumen de texto

Probar cómo Claude puede resumir documentos largos en puntos clave.

### 2. Organización de tareas

Usar Claude para convertir notas desordenadas en una lista clara de acciones.

### 3. Redacción profesional

Pedir a Claude que mejore correos, informes o comunicaciones formales.

### 4. Apoyo al estudio

Crear esquemas, preguntas tipo test y resúmenes para preparar temas.

## Estado del proyecto

Proyecto inicial de aprendizaje y experimentación con herramientas de IA.
